# SpatialCorp Website

Landing page for **SpatialCorp — Unique Audio Solutions**.

🌐 **Live site**: [spatialaudioclub.com](https://spatialaudioclub.com)

## Deployment

This site is deployed automatically via **GitHub Pages** from the `main` branch.
Any push to `main` will update the live site within a few minutes.

## Editing

- Edit `index.html` directly on GitHub or open a Codespace for live preview
- Replace `logo.png` to update the logo
- To change the contact email, search for `CHANGE EMAIL ADDRESS HERE` in `index.html`

## Domains

- **Primary**: spatialaudioclub.com (GitHub Pages)
- **Redirect**: spatialdeejay.com → spatialaudioclub.com (configured via Squarespace)
